Uso de formularios

Los formularios en las paginas web se ocupan principalmente para la obtencion de informacion
de los usiarios para almacenarlos en una base de datos y llevar un orden en cuento a eso se refiere


Cada secuencia en el siguiente formulario se ocupan para un tipo de registro diferente, algunos solo
permiten un tipo de texto en especifico y otras se ocupan para uno o mas registros en una sola sentencia
